This plugin extends WooCommerce with Areeba MasterCard Internet Gateway Service (MIGS).

== Description ==

This plugin uses the redirect method, the user is redirected to Areeba MIGS payment gateway page.

== Installation ==

1. Download the WooCommerce Areeba Migs plugin zip file and extract to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to WooCommerce backend settings page
1. Navigate to Checkout tab
1. Then go to settings of Areeba Migs gateway. Fill the form with credentials and save changes then you are ready.
